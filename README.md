# Jogo da Forca

Projeto de console para jogar Forca, desenvolvido em Java/Kotlin usando Gradle.

## Como executar

1. **Pré-requisitos**  
   - Java 17+  
   - Gradle

2. **Compilação**  
   ```sh
   gradle build
Execução
gradle run --args="P A L A V R A"
Substitua "P A L A V R A" pela palavra desejada (cada letra separada por espaço).
